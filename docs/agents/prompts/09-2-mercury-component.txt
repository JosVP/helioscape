File: src/app/features/game-shell/game-shell.component.ts

2. Subscribe to eventBus.planetSelected$ in ngOnInit:
   - If emitted id === 'mercury' → set activeView('mercury')

3. Add openPlanetPanel(id: string) method that sets activeView/selectedPlanetId.

File: src/app/features/game-shell/game-shell.component.html

Use [style.display] binding — NOT @if — so the Three.js scene is NOT destroyed on switch:

  <app-orrery [style.display]="activeView() === 'orrery' ? 'block' : 'none'" />
  <app-mercury [style.display]="activeView() === 'mercury' ? 'block' : 'none'"
