2. Zone selection
   - Present the zone-select dialog (with the minimap from 25-3) to choose a starting zone from mercury-map.json startingZones. Selecting sets the active build area / centres the viewport there.

3. Guided placement of the initial buildings
   - Step the player through placing the required starting buildings (read the set + order from JSON — e.g. landing pad → power cell → refinery). 4. Resilience
   - If the player reloads mid-flow, resume at the correct step from saved state. No setTimeout-driven steps; progression is state-driven.
