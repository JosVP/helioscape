─────────────────────────────────────────────────────────────────────────────
File: src/app/features/mercury/mercury-hud/zone-select/zone-select.component.ts
─────────────────────────────────────────────────────────────────────────────

Selector: app-mercury-zone-select
Standalone, OnPush.
Inject: DataService, GameStateService.

No inputs. No outputs. Interacts via GameStateService directly.
NOTE: This component renders as a blocking overlay — it has no "cancel" path.
      The player MUST choose a zone before interacting with the map.

Signals:
  readonly zones = signal(this.data.getAllMercuryStartingZones())
  readonly hoveredZoneId = signal<string | null>(null)

Template:
  <div class="zone-select" role="dialog" aria-modal="true" aria-label="Choose your starting zone">
    <div class="zone-select__backdrop"></div>
    <div class="zone-select__content">
      <h2 class="zone-select__title">Choose your starting zone</h2>
      <p class="zone-select__subtitle">
        This choice is permanent. It determines which ore deposits and buildings
        are available from the start.
      </p>

      <div class="zone-select__cards">
        @for (zone of zones(); track zone.id) {
          <div class="zone-select__card"
               [class.zone-select__card--hovered]="hoveredZoneId() === zone.id"
               (mouseenter)="hoveredZoneId.set(zone.id)"
               (mouseleave)="hoveredZoneId.set(null)"
               (click)="confirmSelection(zone.id)"
               role="button" tabindex="0"
               (keydown.enter)="confirmSelection(zone.id)"
               (keydown.space)="$event.preventDefault(); confirmSelection(zone.id)"
               [attr.aria-label]="'Choose ' + zone.displayName">
            <div class="zone-select__card-name">{{ zone.displayName }}</div>
            <div class="zone-select__card-tagline">{{ zone.tagline }}</div>
            <div class="zone-select__card-desc">{{ zone.description }}</div>
          </div>
        }
      </div>
    </div>
  </div>

Methods:
  confirmSelection(zoneId: string): void
    this.gameState.selectMercuryZone(zoneId)
    // Component is removed from DOM via @if in mercury.component.ts after state change

ACCEPTANCE CRITERIA
─────────────────────────────────────────────────────────────────────────────

- On first Mercury visit (mercurySelectedZone === null), the overlay appears
- Map is visible but non-interactive behind the modal (pointer-events: none on canvas when overlay active — add [attr.inert]="gameState.mercurySelectedZone() === null" on <app-mercury-grid>)
- Three zone cards show name, tagline, and description from mercury-map.json
- Clicking a card calls selectMercuryZone(), the modal disappears, map becomes interactive
- Choice persists through save/load (serialised in Block 9.6)
- ng build passes with zero errors
