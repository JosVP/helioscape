Read AGENTS.md (placeholder assets, debug service Block 15) and docs/agents/ARCHITECTURE.md (Canvas components; isometric utils TILE_W=64/TILE_H=32). Owned by:

   - Add a no-build flag to tiles/slots in mercury-map.json (e.g. deep shadow, rough terrain). Placement validation (25-1) must reject these (add a "cannot build here" reason). Render them with a clear visual cue (and in the debug overlay).

PATTERNS / CLEANUP
