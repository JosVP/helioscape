## EARTH — TWO BRANCHES (v0.4)

### DESIGN PRINCIPLE
Naturalist: reduce footprint, nature rebounds, people emigrate.
Architect: stay and grow, efficiency over space, Earth as optimised
civilisation engine. Neither wrong. Both valid at any ratio.
Early game: RP scarcity forces lean one way. Late game: capacity
high enough to fill both. Tagging tracks ratio at First Era Complete.

### SHARED / NEUTRAL NODES (both paths)
- Advanced Renewables Integration — req: Mercury launched
- Dome Habitat Technology — req: Mercury launched
- Deuterium Extraction from Seawater — req: Renewables done
- Fusion Ignition Theory — req: Deuterium done
- Automated Food Systems — req: Arcology Framework OR
  Renewables done. FORK on unlock: choose Rewild freed land
  (Naturalist CE, better organism candidates) OR Develop freed
  land (Architect CE, population + RP boost)

### NATURALIST BRANCH (existing, updated)
- Ocean Macro-Plastic Cleanup — req: Mercury phase 1
- Atmospheric CO₂ Drawdown — req: venus_carbonate spillover
  + first 2 naturalist nodes done (soft gate)
- Soil Restoration Network — req: mars_soil spillover
  + first 2 naturalist nodes done
- Ocean Acidification Reversal — req: venus_carbonate
  + CO₂ Drawdown done
- Rewilding at Scale — req: mars_coastal + Soil done
- Advanced Circular Economy — req: Rewilding at Scale + Fusion Ignition Theory done (mid-late game). Earth's decommissioned industrial zones, waste streams, and junkyard infrastructure processed via fusion-powered atomic sorting systems. All Earth-based construction from this point uses 90%+ recycled material — passive permanent bonus, not a recurring player decision. CE: *"The Last Quarry"* — the final virgin extraction site on Earth is closed; a narrator note about what it feels like when the last wound closes. Small RP capacity boost + minor organism quality bonus for seeding (Earth ecosystems visibly recovering from centuries of extractive industry). Naturalist tag.

Cascade: better pioneer organism candidates for Mars/Venus
seeding. Colonist psychological baseline improved for
open-environment living. Spillover nodes feel earned.

Spillover gate: if player has zero naturalist nodes when
spillover fires, notification appears but node stays locked.
Requires first 2 naturalist nodes before spillover unlocks.
Narrative reason given in notification.

### ARCHITECT BRANCH (new)
- Vertical Megacity Initiative — available early (same tier as
  Renewables). Consolidates population. Raises RP ceiling.
  Frees peripheral land for development not nature. CE: first
  city hits 50M within boundary.

- Arcology Framework — req: Vertical Megacity. Self-contained
  city ecosystems. Reduced resource throughput per capita.
  Unlocks dense_living tag → benefits Mars dome colonists +
  Venus sky city habitability.

- Subsurface City Expansion — req: Arcology Framework.
  Cities expand downward. Less surface footprint. Direct
  cascade: Mars subsurface dome design improved, radiation
  shielding research transfers. CE chain: people who
  prefer underground.

- Planetary Resource Grid — req: Arcology Framework (mid-game).
  Real-time planetary resource optimisation. Small permanent
  RP efficiency bonus. CE: grid prevents first famine before
  it starts.

- Geoengineering Coordination Bureau — req: Planetary Resource
  Grid. Active weather/climate management. Architect tag.
  Research insights transfer to terraforming timeline accuracy.
  Thematic echo: managing a planetary atmosphere on Earth
  while doing the same on Venus.

- Neural-Digital Integration Research — req: Planetary Resource
  Grid + Fusion Ignition Theory (late). Human-system
  integration. RP ceiling increase + extended active lifespans.
  Major philosophical CE chain. Naturalist communities push
  back. Affects Fermi Silence options if active.

- Planetary Coordination Network — req: Neural-Digital
  (endgame capstone). Full automation + AI-assisted logistics.
  Significant late RP ceiling increase. Earth runs as designed
  system. CE chain: what humans do when the planet manages
  itself. Architect endpoint equivalent to Rewilding at Scale.

Cascade: higher RP ceiling earlier. Dome/sky city efficiency
improved. Dense living research benefits colonists in early
Mars domes + Venus sky cities. Subsurface research reduces
Mars surface infrastructure cost pre-atmosphere.

### TENSION SUMMARY
Same node both paths want: Automated Food Systems (fork).
Mutual exclusivity is temporal not permanent — early RP
scarcity forces prioritisation, not permanent lockout.
Some colony benefits mutually exclusive in flavour:
dense_living tag (Architect) vs open_environment bonus
(Naturalist) — both offer colonist benefits in different
contexts.
